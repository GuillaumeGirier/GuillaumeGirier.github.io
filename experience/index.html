PROJECTS

2022 : EITN Fall School in Computational Neuroscience 2022 Project (Paris)

TITLE : Can dendrite-bound attentional signals modulate the activity of orientation selective neurons?

WEBSITE : https://eitnschool2022.sciencesconf.org/

SUPERVISORS : Konstantinos-Evangelos Petousakis (Graduate Student - Master)

COLLEAGUES : Ballbé Julien / Calice Margaux / Cascino Federico / Hall Siobhan / Kurban Kerem / Raison-Aubry Laëtitia / Verardo Claudio

PROJECT DESCRIPTION : In this project, we are interested in modeling the attention process via L2/L3 type pyramidal cells. The neurons will be studied individually and the importance of the structure of the neuron in the process will be studied. Several important questions will be addressed: how to represent the visual type input entering the neuron? How to represent the problem of attention to a stimulus? What are the important parameters to have a working model?

2018 : Physiology of systems project

TITLE : Master first year solo project.

LOCATION : Université Paris-Saclay (Paris) / Sorbonne Université (Paris)

SUPERVISORS : Hédi Soula (PhD, Professor at Sorbonne University) / Lorette Noiret (Lecturer Sorbonne University)

PROJECT DESCRIPTION : The goal is to study the regulation of blood glucose (glucose level in the blood) during an oral intake of glucose. Compared to previous models it is necessary to add a compartment that is that of the intestine. Most of the time this compartment is relatively easy to model but not when dealing with metabolic diseases involving the intestine. The oral glucose tolerance test is a standard test performed to test the body's tolerance to glucose - and thus to detect diabetes. In this project, using what you have learned - in modeling but also in computer science - you will develop a model of blood glucose prediction according to the dose of glucose ingested.
